## 🚀 كيف يعمل البوت الآن؟

1.  ⏰ **كل 6 ساعات** (أو يدوياً)، يقوم GitHub Actions بتشغيل `bot.py`.
2.  💓 يرسل البوت **"رسالة نبض"** تؤكد أنه حي، مع عدد الخلاصات النشطة.
3.  📖 يقرأ ملف `feeds.txt`، ويجلب أحدث 3 مقالات من كل خلاصة.
4.  📬 يُرسل المقالات إلى تيليجرام بتنسيق HTML أنيق مع الصور المصغرة.
5.  💤 ينتهي التشغيل. لا حاجة لأي خادم أو تكلفة.

## ⚡ طريقة الاستخدام (للمالك فقط)

1.  **أضف روابط الخلاصات** في ملف `feeds.txt` (رابط واحد في كل سطر). مثال:
https://feeds.bbci.co.uk/arabic/rss.xml

2.  **احفظ الملف**. هذا كل شيء! سيتولى GitHub Actions الباقي.

3.  **للتشغيل اليدوي**: اذهب إلى تبويب **Actions** > **RSS Bot Check** > **Run workflow**.
