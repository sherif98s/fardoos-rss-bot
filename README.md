# 🗞️ قارئ RSS – Fardoos RSS Bot

> **جندي RSS استراتيجي.** بوت تيليجرام مجاني، يعمل 24/7 على السحابة، يتابع خلاصات RSS ويُرسل أحدث المقالات تلقائياً كل 6 ساعات بتنسيق HTML أنيق مع صور مصغرة.

## 🌟 لماذا هذا البوت؟

| الميزة | الوصف |
|:---|:---|
| 🤖 **ذكاء استباقي** | يفحص الخلاصات تلقائياً كل 6 ساعات ويُرسل الجديد فقط |
| 🧠 **معاينة ذكية** | أمر `/test` لفحص أي خلاصة قبل إضافتها |
| 🎨 **تنسيق HTML** | عناوين عريضة، روابط، تواريخ، وأيقونات |
| 🖼️ **صور مصغرة** | يستخرج الصور تلقائياً من المقالات |
| 💾 **قاعدة دائمة** | تخزين على Railway Volume، لا يفقد البيانات أبداً |
| 🪶 **خفيف ومجاني** | يعمل على خطة Railway + cron-job.org |

---

## 🚀 الأوامر المتاحة

| الأمر | الوظيفة |
|:---|:---|
| `/start` | تشغيل البوت |
| `/add <رابط>` | إضافة خلاصة RSS |
| `/test <رابط>` | معاينة أحدث 3 مقالات من الخلاصة |
| `/list` | عرض الخلاصات المضافة |
| `/check` | فحص فوري للخلاصات |
| `/help` | عرض المساعدة |
